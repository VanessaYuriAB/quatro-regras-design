# Quatro Regras de Design

> Projeto educacional desenvolvido durante os capítulos iniciais da Sprint 1 do curso de
> Desenvolvimento Web da TripleTen Brasil.

Este projeto consiste na criação de uma **página de destino (landing page)** apresentando
quatro princípios básicos de design, seguindo um **roteiro guiado** fornecido pela
plataforma do curso.

Ele serviu como **base estrutural e conceitual** para projeto posterior, o projeto
independente da Sprint sobre Introdução ao HTML e CSS, **“Sobre Mim”**, que pode ser
acessado [aqui](https://github.com/VanessaYuriAB/sobre-mim).

---

## 🎯 Objetivo do Projeto

- Aplicar conceitos fundamentais de **HTML5 e CSS3**
- Aprender a estruturar uma página web completa em uma única tela
- Trabalhar com:
  - imagens de fundo
  - sobreposição (`overlay`)
  - dimensões e posicionamento
  - cartões de conteúdo

---

## 🧠 Conceitos Praticados

- Estrutura semântica HTML:
  - `<header>`
  - `<main>`
  - `<section>`
  - `<footer>`
- Layout com **Flexbox**
- Uso de imagens de fundo (`background-image`)
- Aplicação de **overlay semitransparente**
- Organização de conteúdo em cartões

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
